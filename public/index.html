Click the button above to add ReactionBot to your slack team, an open source chatbot that likes to post reaction images from a link whenever a specific keyword is said. For example, typing the following will tell ReactionBot to post a dog whenever it hears the word "spicy":

@reactionbot add spicy goo.gl/qTc3Re

For a list of commands, message Reactionbot with: "@ReactionBot help"
